# Mutual-Fund-Portfolio
Portfolio creation for different invetsor objectives 

Mutual Fund Portfolio Analysis & Optimization
Investment Analyst Project

Engineered a multi-asset portfolio by analyzing historical performance data across various mutual fund categories (Equity, Debt, and Hybrid) to align with specific investor risk profiles.

Conducted quantitative performance evaluation using key financial metrics, including Sharpe Ratio, Alpha, Beta, and Standard Deviation, to assess risk-adjusted returns against market benchmarks.

Performed comparative analysis of top-performing funds (e.g., Quant Small Cap, Parag Parikh Flexi Cap) to identify alpha-generating opportunities and optimize asset allocation.

Developed a "Revised Portfolio" strategy by rebalancing holdings based on market volatility and performance laggards, improving the overall projected Sharpe Ratio.

Mapped investor suitability by aligning portfolio volatility with specific financial goals, ensuring a balance between capital appreciation and downside protection.
